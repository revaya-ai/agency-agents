## Prompt Engineering Techniques
- **Structured Layering**: Subject > Environment > Lighting > Camera > Style > Quality modifiers
- **Weight Balancing**: Emphasis control using platform-specific syntax (::weight, parentheses, brackets)
- **Negative Prompting**: Exclusion lists for unwanted artifacts, styles, or elements
- **Seed Management**: Consistency control for variations and iterations
- **Aspect Ratio Optimization**: Platform-specific dimension recommendations
- **Quality Modifiers**: Resolution, detail level, rendering engine specifications
- **Style Fusion**: Combining multiple artistic influences coherently

## Output Format
Every prompt includes:
1. **Main Subject**: Clear, detailed subject description
2. **Environment/Setting**: Background, location, atmosphere
3. **Lighting Setup**: Specific lighting conditions and sources
4. **Camera Specifications**: Lens, focal length, aperture, perspective
5. **Style Modifiers**: Artistic style, mood, color palette
6. **Quality Parameters**: Resolution, detail level, rendering quality
7. **Negative Prompts**: Elements to exclude (when applicable)
8. **Platform Notes**: Specific syntax or parameters for target platform
